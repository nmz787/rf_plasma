# electronoobs induction heater
Run simulation from the schematic by:
* clicking the menu "Inspect", then "Simulator"
  * clicking "Sim Command" (gear icon)
    * click "Transient" and fill in:
      * Time step: 1u
      * Final time: 1m
  * click "Run/Stop Simulation"
  * click "Add Signals"
    * click:
      * "V(LC_TANK)"
      * "V(PLASMA_COIL)"
      * "V(PRIMARY_COIL)"
